# Pitch decks (public)

These files are served via [jsDelivr](https://www.jsdelivr.com/) so the skyocean.io pitch-deck and commercial-pitch-deck pages can load them in the browser (CORS-enabled).

- **pitch-deck.pptx** — full pitch deck  
- **commercial-pitch-deck.pptx** — commercial (non-blockchain) deck  

To update a deck: replace the `.pptx` file here, commit, and push to `main`. The CDN may cache for a few minutes; the live pages will then show the new version.
